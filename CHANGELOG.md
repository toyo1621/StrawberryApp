## 1.4.0 - 2026-07-22

- 公開ランキングを表示名ではなく匿名所有者単位へ変更し、同名の別利用者を独立表示
- 所有者情報のない移行前ランキングだけは正規化名単位を維持するD1 migrationと部分インデックスを追加
- 未送信キューからBearerトークンを除去し、旧キュー内の資格情報を読込時に自動洗浄
- 改名後も端末の非公開履歴をオフライン表示できるよう修正
- 名前入力とゲーム開始をランキングより前へ移し、320px初期表示内の主操作をE2Eで保証
- FirefoxとWebKitをE2E・axe検査へ追加し、非初期画面を遅延読込
- D1、Worker、Pagesを同一Git SHAで直列公開し、API待機・公開後スモークでrelease IDを照合
- 本番監視を15分間隔へ短縮し、合成スコアの1件以上削除と削除後不在、任意の外部Webhookを追加
- EAS CLI固定の手動ストアbuild/submit workflowと本番ランキングAPI環境を追加
- 415島データの受領manifestと正規化データセットのSHA-256指紋を記録・自動検証

## 1.3.0 - 2026-07-22

- 415件の有人離島SVGと都道府県・自治体データを追加
- 島モードに日本全国と7地域の出題エリア選択を追加
- 島ランキングを日本全国と7地域に分離し、地域分割前のランキングは関東へ再分類
- 色モードで同じ見た目の2択が出ないラウンド生成へ修正
- API v4でBearer所有者・モード・島地域に結び付く15分有効の使い切りゲームセッションを導入
- サーバー観測時間、得点成立性、D1 batchによるセッション消費とスコア登録を追加
- リモートランキング成功時は端末キャッシュを権威スナップショットへ置換し、削除済み・地域移動済み記録の残留を解消
- 端末履歴を公開ランキングキャッシュから分離し、期限内の保存待ちスコアを起動・アプリ復帰時に同期
- iOS/Androidの秘密トークンをSecureStoreへ移し、旧Async Storageから自動移行
- 島画像の代替テキスト、国旗モードの色コントラスト、4ゲーム・主要画面のaxe検査を改善
- 実Worker/D1 HTTP統合、ソースカバレッジ下限、Hermes/Web容量予算、320px E2Eを追加
- 全GitHub Actionsを不変SHAへ固定し、本番・開発依存のnpm監査を0件へ更新
- Web成果物へAPIオリジン限定CSPとReferrer Policyを追加し、Expo Doctorも固定バージョン化
- D1 Time Travel復元点、Worker release ID、API互換性待機、Web/JS/APIレイテンシ監視を公開手順へ追加
